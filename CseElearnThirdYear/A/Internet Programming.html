CS6501- Internet Programming

COURSE INSTRUCTOR

Dr. B. Bharathi, Associate Professor/CSE Dept

SYLLABUS

Syllabus

LESSON PLAN

Lesson Plan

COURSE OUTCOMES
1.Implement Java programs
2.Design a basic website using HTML and Cascading Style Sheets.
3.Design and implement dynamic web page with validation using Javascript objects and by applying different event handling mechanisms.
4.Design rich client presentation using AJAX and to present data in XML format.
5.Design and implement server side programs using Servlets, JSP, PHP.
